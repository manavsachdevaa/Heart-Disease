# Heart Disease Analysis

This project analyzes heart disease related data using **SQL, data visualization, and web integration** techniques.  
The goal is to identify patterns and relationships between lifestyle factors and heart disease.

The dataset includes health indicators such as BMI, smoking habits, alcohol consumption, physical activity, diabetes, stroke history, age category, and general health.

By combining **SQL data processing**, **Tableau visualizations**, and **Flask web integration**, the project provides insights into how different lifestyle and health factors are associated with heart disease.

---

## Project Objectives

- Analyze health and lifestyle data related to heart disease
- Identify patterns between risk factors and heart disease
- Create visual dashboards for easier understanding of the data
- Present insights through an interactive web interface

---

## Technologies Used

- **SQL** – Data querying and preprocessing  
- **Tableau** – Data visualization and dashboards  
- **Python (Flask)** – Web integration for dashboard display  
- **GitHub** – Version control and project collaboration  

---

## Dataset Features

The dataset includes the following attributes:

- Age Category  
- Sex  
- BMI  
- Smoking  
- Alcohol Drinking  
- Stroke  
- Diabetic Status  
- Physical Activity  
- General Health  
- Heart Disease  

Total records after SQL processing: **4500 rows**

## Visualizations Created

The project includes several visual analyses such as:

- Gender vs Heart Disease
- Age vs Heart Disease
- Diabetic vs Stroke
- Impact of Smoking and Alcohol on Heart Disease
- Stroke vs Other Diseases
- Race-wise Heart Disease
- General Health vs Heart Disease
- Physical Activity vs Heart Disease
- Age vs BMI vs Diabetic
- Stroke with Diabetes and Heart Disease

These visualizations are organized into **3 dashboards and 3 story points**.
